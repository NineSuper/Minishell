## 🎮 Lance notre Minishell !

Copie le projet

```bash
  git clone https://github.com/NineSuper/Minishell
```

Rentre dans le dossier

```bash
  cd Minishell
```

Compile le projet
```bash
    make all
```

Lance l'executable

```bash
  ./minishell
```
